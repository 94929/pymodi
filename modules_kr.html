PyMODI 사용하기

[To English]

입력 모듈

버튼 모듈

  • clicked: 버튼을 눌렀다 떼었을 때에 True를 반환한다.
    • 반환값: True / False
    • 사용 예제: button.clicked

  • double_clicked: 버튼을 더블클릭 했을 때에 True를 반환한다.
    • 반환값: True / False
    • 사용 예제: button.double_clicked

  • pressed: 버튼이 눌렸을 때에 True를 반환한다.
    • 반환값: True / False
    • 사용 예제: button.pressed

  • toggled: 버튼의 상태를 반환한다. 버튼이 눌릴 때마다 반환값이 바뀐다.
    • 반환값: True / False
    • 사용 예제: button.toggled

다이얼 모듈

  • degree: 다이얼의 각도를 반환한다.
    • 반환값: 0 ~ 100
    • 사용 예제: dial.degree

  • turnspeed: 다이얼을 돌리는 속력을 반환한다.
    • 반환값: 0 ~ 100
    • 사용 예제: dial.turnspeed

환경 모듈

  • brightness: 현재 밝기를 반환한다.
    • 반환값: 0 ~ 100
    • 사용 예제: env.brightness

  • humidity: 현재 습도를 반환한다.
    • 반환값: 0 ~ 100
    • 사용 예제: env.humidity

  • temperature: 현재 온도를 반환한다.
    • 반환값: 0 ~ 100
    • 사용 예제: env.temperature

  • red: 현재 빛의 빨강 성분을 반환한다.
    • 반환값: 0 ~ 255
    • 사용 예제: env.red

  • green: 현재 빛의 초록 성분을 반환한다.
    • 반환값: 0 ~ 255
    • 사용 예제: env.green

  • red: 현재 빛의 파랑 성분을 반환한다.
    • 반환값: 0 ~ 255
    • 사용 예제: env.blue

자이로 모듈

  • pitch: y축에 대한 현재 각도를 반환한다.
    • 반환값: -180 ~ 180
    • 사용 예제: gyro.pitch

  • roll: x축에 대한 현재 각도를 반환한다.
    • 반환값: -180 ~ 100
    • 사용 예제: gyro.roll

  • yaw: z축에 대한 현재 각도를 반환한다.
    • 반환값: -180 ~ 100
    • 사용 예제: gyro.yaw

  • angular_vel_x: x축에 대한 각속도를 반환한다.
    • 반환값: -100 ~ 100
    • 사용 예제: gyro.angular_vel_x

  • angular_vel_y: y축에 대한 각속도를 반환한다.
    • 반환값: -100 ~ 100
    • 사용 예제: gyro.angular_vel_y

  • angular_vel_z: z축에 대한 각속도를 반환한다.
    • 반환값: -100 ~ 100
    • 사용 예제: gyro.angular_vel_z

  • acceleration_x: x축 방향으로의 가속도를 반환한다.
    • 반환값: -100 ~ 100
    • 사용 예제: gyro.acceleration_x

  • acceleration_y: y축 방향으로의 가속도를 반환한다.
    • 반환값: -100 ~ 100
    • 사용 예제: gyro.acceleration_y

  • acceleration_z: z축 방향으로의 가속도를 반환한다.
    • 반환값: -100 ~ 100
    • 사용 예제: gyro.acceleration_z

  • vibration: 현재 진동의 정도를 반환한다.
    • 반환값: 0 ~ 100
    • 사용 예제: gyro.vibration

적외선 모듈

  • proximity: 앞의 물체까지의 거리를 반환한다.
    • 반환값: 0 ~ 100
    • 사용 예제: ir.proximity

마이크 모듈

  • frequency: 입력 소리의 진동수를 반환한다.
    • 반환값: float value
    • 사용 예제: mic.frequency

  • volume: 입력 소리의 볼륨을 반환한다.
    • 반환값: 0 ~ 100
    • 사용 예제: mic.volume

초음파 모듈

  • distance: 앞의 물체까지의 거리를 반환한다.
    • 반환값: 0 ~ 100
    • 사용 예제: ultrasonic.distance

출력 모듈

디스플레이 모듈

  • text: 스크린에 문자열을 띄운다.
    • 반환값: 현재 출력되는 문자열
    • 사용 예제: display.text = "Hello World"

  • clear(): 스크린을 전부 지운다.
    • 반환값: None
    • 사용 예제: display.clear()

  • show_variable(): 스크린에 숫자를 표시한다.
    • 반환값: None
    • 인자:
      • variable: 화면에 표시할 숫자
      • position_x: 표시할 세로 방향 위치
      • position_y: 표시할 가로 방향 위치
    • 사용 예제: display.show_variable(2.3, 5, 3) # 2.3을 화면의 (5, 3) 위치에 표시한다.

Led 모듈

  • red: 출력되는 색의 빨강 성분을 바꾼다.
    • 반환값: 현재 색의 빨강 성분
    • 사용 예제: led.red = 255

  • green: 출력되는 색의 초록 성분을 바꾼다.
    • 반환값: 현재 색의 초록 성분
    • 사용 예제: led.green = 255

  • blue: 출력되는 색의 파랑 성분을 바꾼다.
    • 반환값: 현재 색의 파랑 성분
    • 사용 예제: led.blue = 255

  • rgb: 출력되는 색의 rgb 성분을
    • 반환값: 현재 색의 rgb 성분
    • 사용 예제: led.rgb = 255, 10, 30

  • turn_on(): LED 를 최대 밝기로 켠다.
    • 반환값: 255, 255, 255
    • 사용 예제: led.turn_on()

  • turn_off(): LED 를 끈다.
    • 반환값: 0, 0, 0
    • 사용 예제: led.turn_off()

모터 모듈

  • degree: 연결된 두 모터의 각도를 설정한다.
    • 반환값: 두 모듈의 현재 각도
    • 사용 예제: motor.degree = 30, 90

  • first_degree: 모터 I 의 각도를 설정한다.
    • 반환값: 모터 I 의 현재 각도
    • 사용 예제: motor.first_degree = 50

  • second_degree: 모터 II 의 각도를 설정한다.
    • 반환값: 모터 II 의 현재 각도
    • 사용 예제: motor.second_degree = 50

  • speed: 연결된 두 모터의 속력를 설정한다.
    • 반환값: 두 모듈의 현재 속력
    • 사용 예제: motor.speed = 50, 100

  • first_speed: 모터 I 의 속력을 설정한다.
    • 반환값: 모터 I 의 현재 속력
    • 사용 예제: motor.first_speed = 50

  • second_speed: 모터 II 의 속력을 설정한다.
    • 반환값: 모터 II 의 현재 속력
    • 사용 예제: motor.second_speed = 50

스피커 모듈

  • tune: 재생되는 소리의 진동수와 볼륨을 설정한다.
    • 반환값: 현재 진동수와 현재 볼륨
    • 사용 예제: speaker.tune = 220, 100 # 진동수 220Hz 와 볼륨 100의 소리를 재생한다.

  • frequency: 재생되는 소리의 진동수를 설정한다.
    • 반환값: 현재 진동수
    • 사용 예제: speaker.frequency = 550

  • volume: 재생되는 소리의 볼륨을 설정한다.
    • 반환값: 현재 볼륨
    • 사용 예제: speaker.volume = 50

  • turn_off(): 소리를 끈다.
    • 반환값: 0
    • 사용 예제: speaker.turn_off()

설정 모듈

네트워크 모듈

  • button_pressed: MODI Play 의 버튼이 눌렸는지 반환한다.
    • 반환값: 버튼이 눌렸으면 True 아니면 False
    • 사용 예제: network.button_pressed

  • dial: MODI Play 의 다이얼 값을 반환한다.
    • 반환값: 0 ~ 100
    • 사용 예제: network.dial

  • joystick: MODI Play 의 조이스틱 방향을 반환한다.
    • 반환값: 'up', 'down', 'left', 'right', 만약 안 눌렸다면 None
    • 사용 예제: network.joystick

  • left_slider: MODI Play 의 왼쪽 슬라이더 값을 반환한다.
    • 반환값: 0 ~ 100
    • 사용 예제: network.left_slider

  • right_slider: MODI Play 의 오른쪽 슬라이더 값을 반환한다.
    • 반환값: 0 ~ 100
    • 사용 예제: network.right_slider

  • timer: MODI Play 의 타이머가 울렸는지 반환
    • 반환값: 타이머가 울렸으면 True
    • 사용 예제: network.timer

  • buzzer_on(): MODI Play 의 버저를 울리게 한다.
    • 반환값: None
    • 비고만약 작동하지 않는다면 buzzer_off()를 한 후 다시 시도한다.
    • 사용 예제: network.buzzer_on()

  • buzzer_off(): MODI Play 의 버저를 끈다.
    • 반환값: None
    • 사용 예제: network.buzzer_off()

  • take_picture(): MODI Play 로 사진을 찍는다.
    • 반환값: None
    • 사용 예제: network.take_picture()
